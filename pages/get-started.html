OtakuBase API

Welcome to OtakuBase API

OtakuBase API provides endpoints to manage anime data, user reviews, and authentication for your otaku-themed applications.

Features:

  • Fetch anime details like description, genres, and ratings.
  • CRUD operations for anime reviews.
  • User authentication using JWT tokens.
  • Role-based access for Admin and User roles.
  • Upload anime posters to Cloudinary.

Getting Started:

To start using the OtakuBase API, clone the repository and install the dependencies:

git clone https://github.com/aindrilacodes/OtakuBase.git
npm install

Base URL:

https://your-deployed-url.vercel.app

Endpoints:

  • POST /api/auth/register - Register a new user.
  • POST /api/auth/login - Login and receive a JWT token.
  • GET /api/anime/ - Fetch all anime details.
